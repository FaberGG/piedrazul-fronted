# styles

## Proposito
Estilos globales y tokens del sistema de diseno.

## Contiene
- `styles.scss` como punto de entrada.
- Parciales como `_variables.scss` y `_reset.scss`.

## No contiene
- Estilos especificos de un componente puntual.
- Uso indiscriminado de selectores globales o `!important`.

## Referencia
- Basado en `skeleton.md` para `src/styles/`.
