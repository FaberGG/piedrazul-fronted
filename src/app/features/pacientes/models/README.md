# models

## Proposito
Modelos y DTOs TypeScript del dominio `pacientes`.

## Responsabilidad unica
Definir contratos de datos internos del feature.

## Contiene
- Interfaces de entidades de dominio.
- Tipos de request/response para servicios.

## No contiene
- Funciones con efectos secundarios.
- Modelos transversales globales (van en `core/models/`).

## Referencia
- Basado en `skeleton.md` para `src/app/features/pacientes/models/`.
