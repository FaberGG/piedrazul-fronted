# pages

## Proposito
Paginas ruteables del dominio `auditoria`.

## Responsabilidad unica
Coordinar estado, servicios y composicion de UI para cada vista navegable.

## Contiene
- Componentes contenedores por pantalla.
- Interaccion con store y servicios del mismo feature.

## No contiene
- Reutilizacion como subcomponente en otras paginas.
- Logica de acceso HTTP en templates.

## Referencia
- Basado en `skeleton.md` para `src/app/features/auditoria/pages/`.
