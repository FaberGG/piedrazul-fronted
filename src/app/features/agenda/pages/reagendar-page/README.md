# reagendar-page

## Proposito
Pantalla para modificar fecha/hora de una cita existente (RF-08).

## Responsabilidad unica
Representar una unica vista navegable del usuario.

## Contiene
- Composicion de subcomponentes del dominio.
- Eventos y validaciones de la pantalla.

## No contiene
- Dependencias directas de otros features.
- Reglas transversales globales (van en `core/`).

## Referencia
- Basado en `skeleton.md` para `src/app/features/agenda/pages/reagendar-page/`.
