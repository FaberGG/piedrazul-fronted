# tabla-citas

## Proposito
Tabla de citas basada en `shared/components/tabla-datos` con columnas de dominio.

## Responsabilidad unica
Resolver una sola pieza de interfaz dentro del dominio.

## Contiene
- Entradas y salidas para integracion con pages.
- Render de datos ya preparados por servicios/store.

## No contiene
- Navegacion por rutas de primer nivel.
- Llamadas HTTP directas con `HttpClient`.

## Referencia
- Basado en `skeleton.md` para `src/app/features/agenda/components/tabla-citas/`.
