# components

## Proposito
Componentes de UI no ruteables del dominio `medicos`.

## Responsabilidad unica
Encapsular piezas de interfaz del dominio para componer paginas.

## Contiene
- Subcomponentes especializados del dominio.
- Entradas/salidas para comunicacion con pages.

## No contiene
- Dependencias de otros features.
- Responsabilidades de enrutamiento.

## Referencia
- Basado en `skeleton.md` para `src/app/features/medicos/components/`.
