# tabla-datos

## Proposito
Tabla generica configurable para listados con columnas dinamicas.

## Responsabilidad unica
Tabla generica configurable para listados con columnas dinamicas.

## Contiene
- Render de filas/columnas declaradas externamente.
- Soporte de paginacion, carga y estado vacio.

## No contiene
- Columnas fijas acopladas a un dominio.
