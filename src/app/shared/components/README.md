# components

## Proposito
Componentes de presentacion reutilizables, desacoplados del dominio.

## Responsabilidad unica
Componentes de presentacion reutilizables, desacoplados del dominio.

## Contiene
- Componentes con `@Input()` y `@Output()`.
- Comportamiento de UI sin acceso HTTP directo.

## No contiene
- Reglas de negocio de agenda, medicos, etc.
- Inyeccion directa de servicios de feature.

## Referencia
- Basado en `skeleton.md` para `src/app/shared/components/`.
