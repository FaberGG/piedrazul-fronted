# shared

## Proposito
Artefactos reutilizables por dos o mas features.

## Contiene
- Componentes de presentacion genericos.
- Directivas, pipes y validadores compartidos.
- Constantes de dominio comun y UI atomica.

## No contiene
- Logica de negocio acoplada a un unico feature.
