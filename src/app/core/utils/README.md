# utils

## Proposito
Funciones puras reutilizables sin dependencias de Angular.

## Responsabilidad unica
Funciones puras reutilizables sin dependencias de Angular.

## Contiene
- Helpers de fecha/hora para agendamiento.
- Utilidades de roles y formateo de texto.

## No contiene
- Pipes de Angular.
- Funciones que requieran servicios inyectables.

## Referencia
- Basado en `skeleton.md` para `src/app/core/utils/`.
