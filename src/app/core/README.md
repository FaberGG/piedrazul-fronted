# core

## Proposito
Contiene todo lo que existe una sola vez en la aplicacion y se provee en el injector raiz.

## Contiene
- Servicios de autenticacion y sesion.
- Guards e interceptors globales.
- Layouts persistentes de la app.
- Modelos y utilidades transversales.

## No contiene
- Componentes de pantallas de negocio.
- Logica especifica de un feature.

## Referencia
- Basado en `skeleton.md` para `src/app/core/`.
