# src

## Proposito
Codigo fuente de la SPA Angular y sus recursos de ejecucion.

## Contiene
- `app/` con la arquitectura feature-based.
- `environments/` con configuracion por ambiente.
- `assets/` y `styles/` con recursos globales.

## No contiene
- Dependencias de npm o configuraciones de build de raiz.
